# Fang Dong Personal Homepage

This repository powers the GitHub Pages site at:

https://frandium.github.io

The homepage is intentionally static and lightweight:

- `index.html` contains the public page content.
- `styles.css` controls layout and visual design.
- `profile.json` keeps a structured copy of public profile information for future updates.

Do not commit private recruiting notes, interview trackers, phone numbers, or non-public project details to this repository.
