# Fang Dong Personal Homepage

This repository powers the GitHub Pages site at:

https://frandium.github.io

The homepage is intentionally static and lightweight:

- `index.html` contains the public page content.
- `styles.css` controls layout and visual design.
- `translations.js` contains the English and Chinese text shown on the page.
- `assets/profile-photo.jpg` is the homepage portrait.
- `profile.json` keeps a structured copy of public profile information for future updates.

To revise wording or translation, edit the corresponding key in `translations.js`.
The `en` block controls the English version, and the `zh` block controls the
Chinese version. Keep the key names unchanged so the language toggle can still
find the right text.

Do not commit private recruiting notes, interview trackers, non-public contact details, or non-public project details to this repository.
